# 42sh

A TCSH based shell in C

Groups: 4 - 5 People
Time:   4 Weeks

Authorized functions: all functions included in the libC or the ncurses library.

MANDATORY SECTION:
    spaces and tabs,
    $PATH and environment,
    errors and return value,
    redirections (‘<’, ‘>’, ‘<<’ and ‘>>’),
    pipes (‘|’),
    builtins: cd, echo, exit, setenv, unsetenv,
    separators: ‘;’, ‘&&’, ‘||’.

EXTRAS:
    alias
    which / where
    repeat


Execution:
    clone the repo
    
    make

    ./42sh
